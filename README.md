Instruções
=========

## Pacotes necessários: 

* freeglut3 
* freeglut3-dev

## Compilação:


	gcc Main.c -lglut -lGLU -lGL


## Teclas Implementadas:

* '1': Toggle GL_FLAT/GLSMOOTH
* '2': Toggle Wire/Solid
* 'a': Rotação para a esquerda
* 'b': Rotação para a direita
* 'w': Escalonar -
* 's': Escalonar +
* 'espaço': toggle caminhada / corrida
